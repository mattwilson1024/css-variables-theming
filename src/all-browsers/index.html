Theming with CSS variables: All Browsers

This page is styled using CSS variables, click the buttons below to change the theme.

All browsers supported (via ponyfill)

This example uses css-vars-ponyfill, meaning that it even works in IE11!

The element styling can still be defined by referencing CSS variables as usual, but the actual values for these variables are defined and applied in code. css-vars-ponyfill takes care of doing extra fun and games if you're on a legacy browser.
